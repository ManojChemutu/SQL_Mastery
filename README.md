## 🛠 Deploy to GitHub Pages

### Method 1 — Automatic (Recommended)

1. **Fork** or **Use this template** button
2. Go to **Settings → Pages**
3. Under *Source*, select **Deploy from a branch**
4. Choose `main` branch, `/ (root)` folder → **Save**
5. Your site will be live at `https://<your-username>.github.io/k8s-learning-journey/` within ~60 seconds

### Method 2 — Clone & Push

```bash
git clone https://github.com/your-username/k8s-learning-journey.git
cd k8s-learning-journey

# Make your edits to index.html
git add .
git commit -m "feat: customize for my stack"
git push origin main
```

Then enable GitHub Pages in repo Settings as above.
